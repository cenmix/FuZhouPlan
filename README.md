# 浮舟计划 / FuZhou Plan

**版本**: 0.1.0  
**Minecraft**: 1.20.1  
**Forge**: 47.x  
**作者**: FuZhou
**包名**: `org.fuzhou.fuzhouplan`
**模组ID**: `fuzhouplan`
**Forge版本**: 47.4.10

浮舟计划 (FuZhouPlan) 是一款 Minecraft Java Edition Forge 1.20.1 模组，以 生物基因科技 和 工业电力科技 为主要玩法。
欲了解详细信息，请访问: [本模组的 Wiki](https://redersha.github.io/FuZhouPlan_Wiki/)

## 下载

- [Modrinth](https://modrinth.com/mod/fuzhouplan#download)

## 核心玩法

### 生物/基因科技

```
麻醉生物 -> 提取基因样本 -> 解析 DNA -> 获得生物 DNA 储存罐
```

### 工业科技

需要电力系统作为基础。

* 分子材料工业
